# 📒 Weather App

This project is an asynchronous weather app that I made for my JavaScript Certification from FreeCodeCamp. Allowing the execution of multiple threads during the same time. In this weather app it allows you to interact with the site while it waits for the FreeCodeCamp weather API to send back the information about the weather in a few cities. And if the fetch comes back with an error then it will alert the user that something went wrong, which is the case with selecting Paris. Next if we do get our city back, then I used destructuring on the cityWeather object after making it JSON again. Lastly using my variables I assigned with my HTML elements to display the weather information.
